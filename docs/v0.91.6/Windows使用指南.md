## 推荐使用 Web 版本
Recommended Web Version

从 v0.91.1 版本开始，我们不再提供 Windows 桌面版本。

Starting from v0.91.1, Windows desktop version is no longer provided.

**推荐使用 Web 版本**：
**Recommended to use Web version**:

- 🌐 打开浏览器，直接运行 web/index.html
  Open browser and run web/index.html directly

- 📱 支持安装为 PWA 应用，体验类似桌面应用
  Support installing as PWA app, experience similar to desktop app

- 🔄 自动更新，无需重新下载安装
  Auto-update, no need to re-download and install

## Web 版本使用方法
Web Version Usage

### 方法一：直接打开
Method 1: Open Directly

1. 进入 web 目录
   Go to web directory

2. 双击 `index.html`
   Double-click `index.html`

### 方法二：使用本地服务器（推荐）
Method 2: Use Local Server (Recommended)

1. 进入 web 目录
   Go to web directory

2. 双击 `启动Web页面.bat`
   Double-click `启动Web页面.bat`

3. 浏览器自动打开 http://localhost:8000
   Browser opens http://localhost:8000 automatically

### 方法三：安装为 PWA
Method 3: Install as PWA

1. 使用 Chrome/Edge 浏览器打开 Web 版本
   Open Web version with Chrome/Edge browser

2. 点击地址栏右侧的安装图标
   Click install icon on the right side of address bar

3. 应用将添加到开始菜单和桌面
   App will be added to Start Menu and desktop

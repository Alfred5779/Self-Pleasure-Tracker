## 推荐使用 Web/PWA 版本
Recommended: Use Web/PWA Version

Web 版本具有以下优势：
Web version has the following advantages:

- 🌐 无需安装，直接在浏览器运行
  No installation required, runs directly in browser

- 📱 响应式设计，支持手机和电脑
  Responsive design, supports mobile and desktop

- 🔒 所有数据保存在浏览器本地存储
  All data stored in browser local storage

- 🌙 支持白天/夜晚模式
  Supports day/night mode

- 🌐 支持中英文双语
  Supports Chinese/English bilingual

详细使用方法请参考 [Web 使用指南](Web使用指南.md)。

For detailed usage, please refer to [Web Usage Guide](Web使用指南.md).
